# AI Avatar Videos

Add the AI avatar videos for the hero section here.

## Required Videos:

1. **ai-avatar-light.mp4** - AI avatar video for light mode
2. **ai-avatar-dark.mp4** - AI avatar video for dark mode

Alternatively, you can use the main image files:
- **Main Image Light Mode.png** - Static image for light mode
- **Main Image Dark Mode.png** - Static image for dark mode

## Video Guidelines:
- Format: MP4 (H.264 codec)
- Duration: 10-30 seconds (looping)
- Resolution: 1920x1080 or 1280x720
- File size: Under 10MB for optimal performance
- Should feature an AI avatar or animated visualization

## Image Guidelines (if using static images):
- Format: PNG or JPG
- Resolution: 1920x1080
- File size: Under 2MB
- Aspect ratio: 16:9
